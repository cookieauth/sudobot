# `anime`

Sends a anime picture from the desired category into the current channel.

### Syntax

```
-anime [category]
/anime category: [category]
```

### Required Permissions

`NONE`

### Modes

<Modes modes={["legacy", "interaction"]} />

### Examples

```
-anime waifu
/anime category:Neko
```
